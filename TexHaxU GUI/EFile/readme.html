TexHaxU GUI
作成者:upunushiMK78(USH_MK78)

このTexHaxU GUIは、NWPlayer123氏のTexHaxUを使いやすいように改良したものになります。

基本的な使い方
TexHaxUフォルダの中に編集したいszsファイルを入れます。
TexHaxU GUI.exeを起動します。
[yaz0toolを起動]ボタンをクリックします。
コマンドウィンドウが開くので表示されている内容にしたがってbfresファイルに変換します。
[GTXを抽出]ボタンをクリックしてGTXファイルを抽出します。
[GTXをコピー]ボタンを押すとOutDDS_LosslessフォルダにGTXファイルをコピーしてくれます。
[DDSに変換]ボタンをクリックするとGTXからDDSファイルに変換されます。

TexHaxUの設定
bfresファイル名:BFRESファイルの名前を入力します。(OKボタンを押すとFilelist.batが実行されます)
DDSファイル名:編集したDDSファイルの名前を指定します。(TexHaxUフォルダ内にDDSファイルを入れること)
ミップ番号(filelist):番号を入力します。(デフォルトは0)
ミップ番号(splicer):Filelist.batに表示される番号を指定します。

その他
[フォルダを開く]ボタンを押すとOutDDS_Losslessフォルダが開きます。
readme.htmlを開く:クリックするとこのhtmlが開きます。
[yaz0toolを起動]ボタンをクリックするとSZSからBFRESへの変換、BFRESからSZSへの変換をすることができます。
その他:クリックするとTexHaxU GUIの情報を見ることができます。
[GTXとDDSを削除]ボタンを押すと、ConvertフォルダとOutDDS_Losslessフォルダの中身を削除することができます。
